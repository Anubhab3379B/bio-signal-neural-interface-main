# Jupyter Notebooks Directory

This directory is for Jupyter notebooks that provide interactive exploration of the system.

## Suggested Notebooks

1. **signal_preprocessing_exploration.ipynb**
   - Interactive exploration of different preprocessing techniques
   - Visualization of filter effects
   - Feature extraction experiments

2. **intent_classification_analysis.ipynb**
   - Train and evaluate intent classifiers
   - Visualize confusion matrices
   - Experiment with different architectures

3. **adaptive_control_comparison.ipynb**
   - Compare PID vs Neural controllers
   - Visualize control trajectories
   - Stability analysis

4. **sensory_encoding_demo.ipynb**
   - Demonstrate VAE compression
   - Visualize spike trains
   - Reconstruct encoded signals

## Creating Notebooks

To create interactive notebooks:

```bash
# Install jupyter if not already installed
pip install jupyter

# Launch Jupyter
jupyter notebook
```

Then create new notebooks in this directory to explore the system interactively.
